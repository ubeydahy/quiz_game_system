#Quiz Game System

##Project Description
The Quiz Game System is a command-line Python application that allows users to take interactive quizzes on different topics. The system loads questions from a file, presents them to the user one at a time, tracks their answers, calculates their score, and displays their final results.

##Features
### Partner A
- Create and manage a question bank with at least 20 quiz questions
- Save questions to a text file
- Load questions from the text file
- Add new questions through terminal input
- Filter questions by difficulty level (easy, medium, hard)

##Step-by-step Instructions To Run The Project Locally
1. Clone the repository
2. Navigate into the project folder
3. Run the program
4. Follow the instructions in the terminal to start the quiz

### Partner B
-Load quiz questions from the text file created by Partner A
-Shuffle the questions randomly using random.shuffle() so each quiz attempt is different
-Allow the user to select how many questions they want to answer (5, 10, 15, or all)
-Display quiz questions one at a time with four multiple-choice options
-Validate user input to ensure only A, B, C, or D answers are accepted
-Track the number of correct and incorrect answers during the quiz
-Display a results summary showing total attempted questions, correct answers, incorrect answers, percentage score, and grade 
-Show a review of all incorrectly answered questions with their correct answers
-Save the player's name and score to a leaderboard file

##Step-by-step Instructions To Run The Project Locally
1. Clone the repository
2. Navigate into the project folder
3. Run the quiz engine program
4. Follow the instructions in the terminal to select the number of questions and answer the quiz
5. View the results summary and leaderboard after completing the quiz
